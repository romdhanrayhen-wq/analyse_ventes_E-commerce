# analyse_ventes_E-commerce
Projet Python d'analyse des ventes : calcul du CA brut/net, TVA et graphiques avec Matplotlib
Analyse des ventes E-commerce

Description

Ce projet Python permet d’analyser des données de ventes issues d’un système e-commerce. Il calcule des indicateurs clés tels que le chiffre d’affaires brut et net, la TVA, et propose des analyses par catégorie, genre et date. Des visualisations sont également générées pour faciliter l’interprétation des النتائج.

Objectifs

- Automatiser le traitement des données de ventes
- Extraire des indicateurs financiers pertinents
- Fournir des visualisations claires pour l’analyse

Technologies utilisées

- Python 3
- Pandas
- Matplotlib

Structure du projet

- main.py : script principal d’analyse
- data/ : dossier contenant les fichiers de données (non inclus)
- README.md : documentation du projet

Fonctionnalités

- Chargement et nettoyage des données
- Calcul du chiffre d’affaires brut et net
- Calcul de la TVA
- Agrégation des données par catégorie, genre et date
- Génération de graphiques

Installation

1. Cloner le dépôt :
   git clone https://github.com/RayhaneBerzig/analyse_ventes_E-commerce.git

2. Accéder au dossier :
   cd analyse_ventes_E-commerce

3. Installer les dépendances :
   pip install pandas matplotlib

Utilisation

Exécuter le script principal :
python main.py

Données

Le fichier CSV utilisé pour l’analyse n’est pas inclus dans ce dépôt en raison de sa taille supérieure à la limite autorisée par GitHub.
L’utilisateur peut utiliser un fichier similaire ou un échantillon réduit.

Résultats

Le script génère :

- Des indicateurs numériques (CA brut, CA net, TVA)
- Des tableaux agrégés
- Des graphiques représentant les tendances des ventes
